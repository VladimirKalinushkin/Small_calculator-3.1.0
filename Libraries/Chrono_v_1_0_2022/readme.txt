Класс Date находится в пространстве имен Chrono.


Он хранит день, месяц и год даты. (переменные типа int).
Для корректной установки дня недели также есть член Номер Дня. предусмотрен член - день недели (Понедельник - 0).

Для инициализации предусмотрены конструкторы Date(int, int, int) и Date(long int).
Конструктор по умолчанию инициализирует дату текущей датой.

Есть методы set(), аналогичные конструкторам (int, int, int) и (long int).
Метод now() принудительно устанавливает текущую дату.
Методы add_day(), add_month() и add_year() соответственно сдвигают дату на число лет, месяцев или дней.

Для корректной работы в разных операционных системах предусмотрены статические члены zero_s_year и zero_s_day_of_week (Для linux - 1900 год и 0 день недели)
    Находятся в закрытой части класса (private).

Для даты устанавливается текущий день недели.
Есть методы NextWorkDay(). IsWorkday() и WeekOfYear() соответственно для вывода следующего рабочего дня,  
    проверки, является ли теккщий день недели рабочим и указания недели года.

Перегружены операторы ввода и вывода cin >> Date, cout << Date.
Предусмотрены перегруженные операторы ==, !=, <=, <, >= и >.


Технические детали:

    Для установки порядкового номера дня предусмотрен блок четырех функций: get-//-_length (нулевого столетия, 
        если нулевой год - не нулевой в веке, тогда нулевое столетие заканчивается ближайшим нулевым годом столетия (для 1970 го года это 2000 год),
            всех столетий, не считая нулевого и последнего,
            длительности последнего столетия 
                (для 2005 года - последних пяти лет с 2000 по 2004, для 2000 года не вычисляется,)
            и дляительности последнего года).
        Длительностью последнего месяца считается день - просто суммируем.
    Отсчет начинается с нулевого дня, поэтому после вычислений декрементируем номер дня.

    Из-за этого-же перед установкой даты из номера дня - set(long int) - номер дня инкрементируется.
Для этого метода предусмотрены функции get_year() и get_month(), обращающиеся к функциям
get-//-_length через соответствующий блок функций (get-//-_in: zero_century, century, last_century).

Длительность месяца записана в статической переменной в функции GetMonthDay().


При обработке исключений вызывается функция error_output(string), член класса Date.
    Кроме вывода сообщения об ошибке она устанавливает для текущего объекта текущую дату.
